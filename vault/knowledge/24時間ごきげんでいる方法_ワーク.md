## ワーク①：4ステップ 認知修正シート

動揺した出来事があったときに記入する。紙に書くか、このファイルに追記して繰り返し使う。

---

### セッション（日付：　　　　　）

**Step 1｜出来事の特定**
自分を動揺させた（ざわざわさせた）事実を1つ書く。

```
（例：上司に仕事の期限を急かされた）

→
```

---

**Step 2｜自動思考の記述**
その瞬間にパッと湧いた感情・頭の中の言葉を書く。

```
（例：「なんで自分だけ？」「不公平だ」「焦り・怒り」）

→
```

---

**Step 3｜合理的思考への変換**
思考の歪み（極端な解釈）を特定し、論理的な見方に書き直す。

| 思考の歪み（気づいたもの） | 修正後の合理的思考 |
|---|---|
| 例：「全部自分のせいだ」 | 例：「自分が担当している部分に集中すれば良い」 |
|  |  |
|  |  |

論理的な対策・次のアクション：
```
→
```

---

**Step 4｜結果の評価**
考え直した後、気分の変化を書く（0〜10の数値でも可）。

```
ワーク前の気分スコア（0=最悪, 10=最高）：
ワーク後の気分スコア：

気づいたこと・変化：
→
```
